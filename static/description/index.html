# Brevo Connector - Odoo Module ## Module Information - **Name**: Brevo Connector - **Version**: 18.0.1.0.39 - **Author**: Florian Neuhuber - **License**: AGPL-3 - **Category**: CRM ## Description This module provides bidirectional synchronization between Odoo and Brevo (formerly Sendinblue) API v3. It enables seamless integration of contact management, list synchronization, and CRM lead creation from Brevo bookings. ## Key Features - Bidirectional contact synchronization - Contact list management - CRM lead creation from bookings - Webhook support for real-time updates - Comprehensive configuration wizard - Detailed logging and monitoring ## Installation Requirements - Odoo 18.0 Community Edition - Python dependencies: brevo-python - Brevo API key ## Installation Instructions 1. Install Python dependencies: `pip install -r requirements.txt` 2. Copy module to Odoo addons directory 3. Update addons list in Odoo 4. Install module from Apps menu 5. Configure via Settings > Technical > Brevo Integration ## Configuration 1. Run the Setup Wizard 2. Enter Brevo API key 3. Test connection 4. Configure webhooks (optional) 5. Set field mappings ## Support For technical support and questions, please refer to the module documentation or contact the development team. ## License This module is licensed under AGPL-3.
